# Agent instructions

このリポジトリでは、2Dゲーム素材を作る。
画像生成や画像編集が必要なときは、必要に応じて $imagegen を使う。

スプライト作成タスクでは必ず `skills/sprite-forge.md` の手順に従う。

成果物は必ず以下に保存する。

- `assets/sprites/<character_id>/raw/`
- `assets/sprites/<character_id>/frames/`
- `assets/sprites/<character_id>/sheets/`
- `assets/sprites/<character_id>/preview/`
- `assets/sprites/<character_id>/prompts/`

完了時は、生成したファイル一覧と、次に改善すべき点だけを短く報告する。
